Create Memberships with special rules for your customers

== Description ==

EASY AND FEATURE RICH MEMBERSHIP SYSTEM FOR WORDPRESS/WOOCOMMERCE

Sell Memberships and give exclusive access and prices to your members. Members can have special prices, see private pages, posts and products.
You can even set multiple VIP Memberships with different rules for each membership.

== WooVIP gives you the possibility to : ==
* **Sell Memberships** : Sell permanent or temporary memberships to your customers.
* **Create unlimited memberships** : Create various levels of memberships, each one with its own privileges
* **Product rules** : Give your members access to exclusive products, set special discounts for them and more ...
* **Product variations** : Set special rules for each variation of a product.
* **Pages and Post rules** : Set which pages and posts are exclusive to members. Other users won't see them.
* **Teaser shortcode** : Tease regular customers with a glimpse of what members can see.
* **Membership expiration** : Inform your customers about their membership status. Send an email when a membership is about to expire.
* **Easy to set up** : With a clean and intuitive interface, you'll easily find the options you're looking for.
* **Roles management** : Manage the user roles, create or delete new roles used for memberships.
* **WPML Compatibility** - WooVIP is compatible with WPML

== The free version offers the following features : ==
* **Assign VIP membership to users** - Manually give a membership to your users.
* **Page visibility** - Set if a page can only be seen by members or not.
* **Post visibility** - Set if a post can only be seen by members or not.
* **Product visibility** - Set if a product can only be seen by members or not.
* **Product price** - Set special prices on products for members
* **WPML Compatibility** - WooVIP is compatible with WPML

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/woovip` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooVIP screen to configure the plugin


== Frequently Asked Questions ==

= I bought the pro version but still have only the free features, why ? =

After buying the pro version, you must enter your licence key in the *Licence* Tab of your plugin administration.
Once the plugin is activated, you can either click the "Install Pro Version Now" or wait for Wordpress to detect an update available for that plugin.
Update the plugin to get access to the pro features.
